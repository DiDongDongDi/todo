# 交互与手势规范

## 布局

```
┌─────────────────────────────┐
│                             │
│      大卡片（主体内容）        │
│                             │
├─────────────────────────────┤
│   【 收 集 】    【 处 理 】   │  ← 底部 Tab 栏
└─────────────────────────────┘
```

- 底部 Tab 栏，默认选中「收集」
- 大卡片区域占 Tab 栏上方全部空间
- 底部预留安全区，减少与上划手势冲突

---

## Tab 1：收集

| 状态 | 手势 | 行为 | 反馈 |
|------|------|------|------|
| 空白 | 上划 | 无效 | 回弹 + 轻提示「先记下点什么」 |
| 有内容 | 上划 | 保存到收集箱 | 卡片飞出 + haptic + toast |
| 任意 | 左/右滑 | 无 | — |

### 录入方式

| 方式 | 操作 |
|------|------|
| 文字 | 点击卡片区域，键盘输入 |
| 语音转文字 | 点麦克风，实时 STT 填入 |
| 贴图 | 粘贴或从相册选择 |
| 纯录音 | 录音后显示波形，后台转写 |

### 保存后

- 卡片重置为空白
- Toast：「已收集，去处理页看看」

---

## Tab 2：处理

| 手势 | 行为 | 去向 | 反馈 |
|------|------|------|------|
| 左滑 | 放弃 | 回收站 | 卡片左飞出 + haptic |
| 右滑 | 完成 | 归档 | 卡片右飞出 + haptic |
| 上划 | 下一条 | 仍在队列 | 卡片上滑切换 |
| 下划 | 上一条 | 仍在队列 | 卡片下滑切换 |
| 点击 | 进入编辑 | — | 内联编辑模式 |

### 空状态

- 无 inbox 任务时显示：「收集箱是空的，去收集页记一条吧」

### 撤销

- 左滑/右滑后底部 Snackbar 3 秒可撤销

---

## 收集 vs 处理 手势对照

| | 收集 | 处理 |
|--|------|------|
| 上划 | 保存（有内容时） | 下一条 |
| 下划 | — | 上一条 |
| 左滑 | — | → 回收站 |
| 右滑 | — | → 归档 |
| 编辑 | — | ✓ |

---

## 桌面 / Web 快捷键

### 收集 Tab

| 按键 | 行为 |
|------|------|
| `Enter` | 保存（有内容时） |
| `Esc` | 清空当前输入 |

### 处理 Tab

| 按键 | 行为 |
|------|------|
| `←` | 放弃（→ 回收站） |
| `→` | 完成（→ 归档） |
| `↑` | 上一条 |
| `↓` | 下一条 |
| `Enter` / 双击 | 进入编辑 |

### 全局

| 按键 | 行为 |
|------|------|
| `Cmd/Ctrl+1` | 切换到收集 |
| `Cmd/Ctrl+2` | 切换到处理 |

---

## 动效原则

- 所有滑动手势 **跟手**，松手后根据阈值决定执行或回弹
- 滑出动画 200–350ms，ease-out
- 庆祝动画 < 1s，可跳过
- 切换 Tab 时大卡片可有轻微淡入

---

## 首次使用引导

1. 收集 Tab：空白卡片 + 提示「写下一件事，上划保存」
2. 处理 Tab：首张卡片演示左右滑方向色带（左红放弃、右绿完成）
